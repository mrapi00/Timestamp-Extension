# Timestamp Chrome Extension

A Google Chrome extension which allows users to save timestamps of a YouTube video as they view it, and playback the video at the timestamp.

![alt text](https://mahmudulrapi.netlify.app/extension.0b1b0a13.png)

### To load my Chrome Extension
``` sh
git clone https://github.com/mrapi00/Timestamp-Extension
```

Then head over to chrome://extensions/ and click on
Load unpacked to add the Timestamp-Extension folder.
